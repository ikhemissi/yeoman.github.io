How to add your generator »

Your generator will automatically show up in the list if it meets the following criteria. The list is populated every hour.

  • Be published on the npmjs.org repository
  • Have the yeoman-generator keyword
  • Have a description in the package.json file or in its associated Github repository

It'd also be nice if your package.json has the following information:

  • Author information including name and link to your homepage
  • Generator documention link as its homepage (or at least the Github link to its repo)
